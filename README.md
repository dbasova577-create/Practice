# Модуль расчета стоимости отеля и автоматические тесты

Данный репозиторий содержит программный модуль калькулятора для расчета стоимости проживания в отеле, а также развитый комплекс автоматических тестов (юнит-тесты и параметризованные интеграционные тесты).

Проект разработан в рамках прохождения учебной практики.

##  Технологический стек
* **Язык программирования:** Python 3.13
* **Фреймворк для тестирования:** pytest
* **Анализ покрытия кода:** pytest-cov
* **Формат данных конфигурации:** JSON
* **Непрерывная интеграция (CI):** GitHub Actions (виртуальная среда Ubuntu Linux)

##  Установка зависимостей проекта

Перед запуском проекта локально убедитесь, что у вас установлен Python версии 3.13. Для установки всех необходимых библиотек выполните в терминале команду:

```bash
pip install -r requirements.txt
```

##  Запуск автоматических тестов

Для локального запуска всего набора тестов (включая проверку успешных тест-кейсов из файла и негативных сценариев валидации) выполните:

```bash
pytest -v
```

### Подсчет покрытия кода (Coverage Report)
Для запуска сканирования плотности тестов и генерации финального терминального отчета о покрытии строк кода используйте команду:

```bash
pytest --cov=calculator
```
*Текущий показатель покрытия кода тестами составляет **100%**.*

##  Инфраструктура непрерывной интеграции (CI)
В проекте настроен автоматический пайплайн в **GitHub Actions** (`.github/workflows/tests.yml`). Облачный робот автоматически запускает проверку работоспособности кода и прогон тестов при каждом выполнении команды `git push` в удаленный репозиторий.
